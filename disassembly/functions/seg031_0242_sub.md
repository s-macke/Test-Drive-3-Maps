; sub_190A2 releases a file slot at dseg:C39A (8-byte entries) after flush/close.
; Skips slots with flags clear or with flag 0x40 set (device/console).
; Uses sub_19E98 to flush buffered data and sub_1A832 to close the handle in [si+7].
; If the slot id (word table at dseg:C43E, index=(si-0C39A)/8) is nonzero, builds a delete path:
; - copies "\" via sub_195EA, appends "\" with sub_1ACDA when needed,
; - appends the decimal id via sub_1AD1A (base 10),
; - deletes via sub_1ADBE (int 21h AH=41h).
; File-slot layout (observed):
; - +0/+2/+4 are buffer state fields used by the read/write helpers,
; - +6 is a flags byte (0x83 in-use/mode, 0x40 device),
; - +7 is the DOS file handle.
